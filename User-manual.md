## Getting Started

### Installation

Simply download the `wincu.exe` executable or build it from source:

```powershell
go build -o wincu.exe cmd/main.go
```

### Privileges

Some cleaning tasks (like Windows Update Cache, Prefetch, Windows Temp) require **Administrator privileges**.

- To run as admin: Right-click your terminal (PowerShell or Command Prompt) and select "Run as administrator".

## Commands

### 1. Scan (`wincu scan`)

Analyzes your system for junk files and reports how much disk space can be reclaimed.

**Usage:**

```powershell
wincu scan [flags]
```

**Flags:**

- `--json`: Output the result in JSON format (useful for scripts/automation).

### 2. Clean (`wincu clean`)

Removes junk files from the specified targets.

**Usage:**

```powershell
wincu clean [flags]
```

**Flags:**

- `--all`: Clean all supported targets (User Temp, Windows Temp, Prefetch, Windows Update, Recycle Bin).
- `--temp`: Clean User Temp and Windows Temp folders.
- `--recyclebin`: Empty the Recycle Bin.
- `--prefetch`: Clean Prefetch files.
- `--update`: Clean Windows Update cache.
- `--chrome`: Clean Google Chrome Cache.
- `--edge`: Clean Microsoft Edge Cache.
- `--browser`: Clean all browser caches (Chrome + Edge).
- `--dry-run`: Simulate the cleaning process without deleting anything.
- `--force`: Force delete read-only files. **Auto-Elevation**: If run without admin rights, this flag will trigger a UAC prompt to restart as Administrator.
- `--threads <n>`: Set the number of concurrent deletion threads (default: CPU cores).
- `--json`: Output log messages in JSON format.

## Examples

**Check reclaimable space:**

```powershell
wincu scan
```

**Preview what would be deleted (Safe Mode):**

```powershell
wincu clean --temp --dry-run
```

**Clean temporary files and Recycle Bin:**

```powershell
wincu clean --temp --recyclebin
```

**Full system cleanup (Run as Admin):**

```powershell
wincu clean --all --force
```

**Clean silently with 16 threads (JSON output):**

```powershell
wincu clean --all --threads 16 --json > cleaner.log
```

## Troubleshooting

- **"Permission denied"**: Ensure you are running the terminal as Administrator. `wincu` will skip protected system files if it doesn't have the necessary rights.
- **"File in use"**: Some files cannot be deleted because they are currently open by other programs. `wincu` will skip these safely.
- **"Access is denied" on specific files**: Try using the `--force` flag to handle read-only files.
